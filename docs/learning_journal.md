## Project Architecture

The project will consist of three major components:

1. **Machine Learning**
   - ECG preprocessing
   - Dataset management
   - Model training
   - Model evaluation
   - Explainability

2. **Backend**
   - FastAPI inference service
   - Model loading
   - Prediction API
   - PDF report generation

3. **Frontend**
   - ECG upload
   - ECG visualization
   - Prediction dashboard
   - Confidence scores
   - Explainability visualization
