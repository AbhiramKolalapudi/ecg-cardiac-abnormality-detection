# AI-Assisted ECG Cardiac Abnormality Detection

A production-style Clinical Decision Support System (CDSS) for detecting cardiac rhythm abnormalities from ECG signals using deep learning.

## Planned Features

- ECG preprocessing
- Deep learning model (1D CNN)
- Multi-class arrhythmia classification
- Confidence scores
- Explainable AI
- FastAPI backend
- React frontend
- ECG visualization
- Doctor dashboard
- PDF report generation
